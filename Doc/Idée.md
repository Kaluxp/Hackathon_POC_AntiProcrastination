# “Procrastination Detector”

Très original si bien présenté.

Concept

L’app détecte les comportements de procrastination AVANT qu’ils arrivent.

Exemple :

alt-tab fréquent,
YouTube ouvert,
VSCode inactif,
scrolling rapide,
changements constants d’onglets.

Puis :

“Tu sembles bloqué, on découpe la tâche ?”

Angle intelligent

Tu combats :

le “freeze mental”,
pas juste le manque de discipline.
Tech intéressante
extension navigateur,
desktop app,
analytics comportementales.

Très bon pour impressionner un jury

## “Anti-Procrastination for Devs” — le copilote d’inertie

Le problème des devs n’est souvent pas :

“je sais pas quoi faire”,
mais :
“je sais pas par où commencer”.
Concept

L’app connecte :

GitHub,
Jira/Trello,
VS Code,
éventuellement Discord.

Et transforme :

“Finir le backend auth”

en :

“ouvre auth.js”
“corrige cette route”
“teste ce endpoint”

L’IA découpe automatiquement les tâches.

## “The Freeze Breaker”

Le problème ciblé :

le moment exact où quelqu’un freeze devant une tâche.

L’app détecte :

hésitation,
inactivité,
changements de tabs,
scroll inutile,
ouverture/fermeture répétée.

Puis :

découpe automatiquement,
donne UNE action,
crée un mini challenge.

C’est très moderne et différent.

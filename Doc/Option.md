# Option

    Padawan (Pomodoro)
    Jedi Focus (signature)
    Sith Hardcore (fun)

## Pomodoro classique

### Fonctionnement

    25 min travail
    5 min pause
    4 cycles → grande pause
    TIMER

### Récompense
    XP stable
    progression équilibrée

### Pour
    débutants
    étudiants
    gens qui procrastinent beaucoup

## Jedi Focus

### Fonctionnement
    temps libre
    pause gagnée selon activité

    Exemple :
    10 min focus = 2 min pause

### Récompense
    bonus de discipline
    rang monte plus vite

### Pour
    devs expérimentés
    gros focus sessions

## Mode Sith Hardcore
### Fonctionnement
    aucune pause automatique
    pauses très limitées
    timer strict
### Si
    pause dépassée
    → perte XP/rank
### Récompense
    XP x2
    badges rares
### Pour
    grinders
    compétiteurs
    hackathons

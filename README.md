# Hackathon_POC_AntiProcrastination
# 🚀 Focus Extension (VS Code)

Extension VS Code pour lancer des sessions de travail avec timer (anti-procrastination).

---

## 📦 Installation

1. Cloner le repo :

```bash
git clone <repo-url>
cd <repo>
```

Installer les dépendances :
```bash
npm install
```
▶️ Lancer l'extension
Ouvrir le projet dans VS Code :
```bash
code .
Appuyer sur :
F5
```

👉 Cela ouvre une nouvelle fenêtre VS Code avec l’extension active
Utilisation
Ouvrir la palette de commande :
```bash
Ctrl + Shift + P
```
Taper :
```bash
Start Focus Session
```
Lancer une session de travail
